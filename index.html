Communication and Computer Networks

Network - A group of 2 or more similar things or people interconnected with other.
Data in a network is divided into smaller chunks called packets.

Components of Computer Network

  1. Hosts / Nodes
  2. Servers
  3. Clients
  4. Network Hardware
  5. Communication Channel
  6. Network Services
  7. Software

  1. Hosts - The computers that are attached to a network.Also known as WORKSTATION.

  2. Server - Facilitates networking tasks like sharing of data , resource shaing , communication aming hists etc.

  3. Client - Host computer that request for some services from a server.

  4. Network Hardware - Requires specialised hardware to carry out different roles / jobs.

  5. Communication Channel - The path through which message travel between source and destination. Also known as Transmission Medium or link which is either wired or wireless.

  6. Software - Makes networking possible.

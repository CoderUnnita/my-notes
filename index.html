Communication and Computer Networks

Network - A group of 2 or more similar things or people interconnected with other.
Data in a network is divided into smaller chunks called packets.

Components of Computer Network

  • Hosts / Nodes
  • Servers
  • Clients
  • Network Hardware
  • Communication Channel
  • Network Services
  • Software

  • Hosts - The computers that are attached to a network.Also known as WORKSTATION.
  • Server - Facilitates networking tasks like sharing of data , resource shaing , communication aming hists etc.
  • Client - Host computer that request for some services from a server.
  • Network Hardware - Requires specialised hardware to carry out different roles / jobs.
  • Communication Channel - The path through which message travel between source and destination. Also known as Transmission Medium or link which is either wired or wireless.
  • Software - Makes networking possible.
